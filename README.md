## Istallazione

MaterialDesign is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'MaterialDesign', :git => 'https://github.com/mattiaberretti/MaterialDesign.git'
```
